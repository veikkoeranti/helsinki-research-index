Miért bíznak a finnek a rendőrségben? Új rendészeti megoldások Finnországból

OpenAlex · DOI · hu

Abstract

Using the case study of the virtual community policing Itäkeskus pilot project in Helsinki, the author shows the successful Nordic model for community policing, which combines the best criminological statistics and the lowest number of officers employed.
